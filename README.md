# Graphql-server

## Installation

```
git clone git@github.com:abhisheksarmah/graphql-server.git graphql-server

cd graphql-server

cp .env.example .env

php artisan key:generate

composer install
```

## Usage

-   create a database called `graphql`

```
php artisan migrate

php artisan db:seed
```
